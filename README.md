# 🎮 Play With Me - Games Zone Portal

موقع حديث وسريع لتحميل الألعاب الجوال مع واجهة مستخدم جميلة وتجربة سلسة.

## ✨ المميزات

### 🌍 دعم اللغات
- 🇸🇦 العربية
- 🇬🇧 الإنجليزية
- تبديل فوري بين اللغتين مع حفظ التفضيل

### 📱 تصميم استجابي
- تصميم كامل الاستجابة (Desktop, Tablet, Mobile)
- بطاقات الألعاب تظهر بـ 2 في الهاتف
- واجهة سهلة الاستخدام

### 🎯 المميزات الرئيسية

#### 1. الرأس (Header)
- شعار موقع جميل
- قائمة ملاحية مع الرموز
- زر فتح القائمة (Hamburger Menu) للجوال
- زر تبديل اللغة
- زر البحث

#### 2. القائمة (Navigation)
عند فتح القائمة تظهر:
- 🏠 الرئيسية (Home)
- ℹ️ حول (About)
- 🎵 تيكتوك (TikTok)
- 🌍 تبديل اللغة
- 🔍 البحث

#### 3. قسم البحث
- بحث فوري عن الألعاب
- عرض النتائج مباشرة
- الانتقال السريع للعبة المختارة

#### 4. قسم الألعاب
- عنوان "ألعابنا"
- وصف "اختر لعبتك المفضلة وابدأ اللعب الآن"
- عرض 8 ألعاب جديدة

#### 5. بطاقة اللعبة
تحتوي كل بطاقة على:
- 🖼️ صورة اللعبة
- 📝 اسم اللعبة
- 📄 وصف قصير
- ⭐ تقييم بالنجوم
- 📥 زر التنزيل
- ▶️ زر التشغيل (Play Now)

#### 6. نافذة تفاصيل اللعبة (Modal)
عند الضغط على اللعبة أو التنزيل:
- 🖼️ صورة اللعبة الكبيرة
- 📝 اسم اللعبة
- 📄 الوصف الكامل
- ⭐ التقييم
- 🟢 زر Download for Android (أخضر)
- ⚫ زر Download for iPhone (رمادي)

### 🎨 التأثيرات والرسوميات
- ✨ تأثيرات الظل والتدرج
- 🔄 انتقالات سلسة (Transitions)

- 📊 مؤثرات عند التمرير (Hover Effects)
